# Welcome to CheckOnChain!

This repository hosts a number of static charts relating to network valuation and pricing models for [Bitcoin]('btc_charts\btc_charts.md') and [Decred]('dcr_charts\dcr_charts.md').

Charts are updated on a semi-regular basis (1-3day intervals) and will eventually be hosted live and updated daily.

All of these models are based on custom python code and analysis of data sourced from Coinmetrics Community API and dcrdata.
